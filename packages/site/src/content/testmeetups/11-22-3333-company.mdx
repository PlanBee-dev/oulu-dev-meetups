## Reactive coding 101

Learning about reactive programming is crucial for building responsive apps. In this talk, we'll cover:
- What is reactive programming
- The reactive manifesto
- RxJS fundamentals like Observables and Operators

## Useful tools

We'll also discuss some useful reactive tools like: 
- NgRx (state management for Angular)
- React Query (data fetching for React)
- MobX (simple reactive state management)

## Ask us anything!
Join us after the talks for Q&A and networking over snacks and drinks. 
Feel free to chat with our developers about anything coding or AI related. We're excited to meet other local tech enthusiasts!

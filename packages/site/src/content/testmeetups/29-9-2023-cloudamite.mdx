## How to build a meetup site
With Astro, Tailwind, Github pages, Github actions, Octokit and Cloudflare workers. 

## And for testing

- [CoPilot](https://github.com/features/copilot)

Vitest and Playwright are great options for testing Astro sites. Vitest provides simple unit testing while Playwright 
allows end-to-end testing of the full site functionality.


We wanted to create a simple and easy to use site to organize local developer meetups in our town. So come and learn about the whole stack we used.
